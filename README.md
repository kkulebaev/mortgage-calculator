# Mortgage calculator

## App Preview

Demo available on https://calculator-mortgage.netlify.app/

## Installation Steps

- Clone repository `git clone https://github.com/kkulebaev/mortgage-calculator.git`
- Install dependencies `pnpm install`
- Run project `pnpm dev`
